Auto property with private setter

Sometimes I’d like to only allow a property to be set in the constructor. Something like this: private Foo _foo; public Foo Foo { get { return _foo; } } public Bar(Foo foo) { _foo = foo; } This neat little pattern lets you do the same with fewer members (and lines of code): public Foo { get; private set; } public Bar(Foo foo) { Foo = foo; }
